# Static HTML Website Project

This is a lightweight static website project for Lite/single-HTML flows.

- Edit `index.html` directly for the page experience.
- Keep the result self-contained unless a small local asset is clearly needed.
- Use `src/main.ts` only for tiny optional behavior required by the page.
- Do not convert this project to React, Tailwind, shadcn/ui, or a full app scaffold.
- The shared publisher will build with Vite, commit the project, and upload the full project zip.
